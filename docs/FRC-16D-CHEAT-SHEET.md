- **W (Witness):** Meta-awareness magnitude ($W = |\vec{U}_{16}|$). The ability to see the system rather than *be* the system.

## 4. The Four Failure Modes (Diagnostics)

When the vector field destabilizes, it falls into one of four catastrophe types.

### 1. Collapse (Fold)
System shrinks. Low RU, Low $\kappa$.

- **Symptoms:** Depression, freeze, shutdown.
- **Fix:** Grounding, routine, metabolic repair (raise RU).

### 2. Inversion (Pitchfork)
Axis flips sign. $\kappa < 0$.

- **Symptoms:** Self-sabotage, addiction, betraying own values.
- **Fix:** Truth-telling, shadow work, realigning narrative.

### 3. Dissociation (Hopf)
Detachment. High W, Low $\kappa$.

- **Symptoms:** Spiritual bypass, "floating," theory without action.
- **Fix:** Somatics, physical touch, reducing witness dominance.

### 4. Dispersion (Saddle-Node)
Explosion. High RU, Low $\kappa$.

- **Symptoms:** Mania, grandiosity, "too many ideas," scattering.
- **Fix:** Structure, sleep, limiting inputs, stopping expansion.
